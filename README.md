**노아에이티에스㈜ 2026년 신입사원 채용 사전과제**  
**주제:** 9번 - 자동차 vs 대중교통

---

## 프로젝트 개요

"내 상황에서 차를 사는 것이 합리적인가?"

사회초년생이 차량 구매 여부를 경제적으로 판단할 수 있도록 지원하는 웹 애플리케이션입니다.  
감가상각, 유류비, 보험, 세금, 정비비, 그리고 **통근 시간의 기회비용**까지 포함한 총비용(TCO)을 계산하여 대중교통과 비교합니다.

**배포 URL:** https://noaats-dongyeon.vercel.app

---

## 핵심 기능

- **대중교통 vs 자동차 월 총비용 비교**  
- **시간 가치(기회비용) 토글** - 포함/제외 선택 가능  
- **손익분기점 계산** - 차량 구매 비용 회수 시점  
- **차트 시각화** - 막대 그래프, 도넛 차트  
- **챗봇 형식 UX** - 단계별 대화형 입력  
- **한국어 자연어 파싱** - "이천오백만원", "2천5백" 인식 (AI 활용)  
- **현실적 시나리오** - 현재 차량 보유, 할부금, 차량 교체 등 반영

## 주요 계산 로직

### 1. 대중교통 비용
```
월 교통비 = 편도 요금 × 2 × 출근일수
월 시간비용 = (편도 시간 × 2 × 출근일수 ÷ 60) × 시급
```

### 2. 자동차 비용
```
월 감가상각 = 구매가 ÷ (사용 기간 × 12)
월 유류비 = (거리 × 2 × 출근일수 ÷ 연비) × 유가
월 고정비 = 보험/12 + 세금/12 + 주차비 + 정비비/12
월 변동비 = 통행료 × 2 × 출근일수
월 시간비용 = (편도 시간 × 2 × 출근일수 ÷ 60) × 시급
```

### 3. 손익분기점
```
월 절감액 = 대중교통 총비용 - (자동차 운영비 - 감가상각)
손익분기점 = 차량 구매가 ÷ 월 절감액
```

## 엣지 케이스 처리

| 케이스 | 처리 방법 |
|--------|----------|
| 시급 "몰라" 입력 | 시간비용 0원, "시간 비용 제외" 안내 |
| 현재 차량 시세 < 할부금 | 음수 순가치 경고, 실제 비용 반영 |
| 현재 차량 유지 | 감가상각 0원, 손익분기점 "해당 없음" |
| 한글 숫자 ("이천오백만원") | Groq AI 파싱 |
| 차량명 입력 ("셀토스") | 가격 직접 입력 요청 |
| 비정상 범위 | 재입력 요청, 대화형 에러 메시지 |
